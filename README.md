# Simple Money Tracking App


## pre-dev

npm install --save react

npm install --save react-dom

npm install --save parcel-bundler


## dev

go run server/main.go

parcel ./ui/src/index.js --public-url /s/

go build -o gmz server/main.go


## run 

./gmz
